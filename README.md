# 📝 Notes API (Node.js + Express + MongoDB)

This is a simple REST API built using Node.js, Express.js, and MongoDB.  
It performs full CRUD operations for managing notes.

---

## 🚀 Features

- Create a note (POST)
- Get all notes (GET)
- Get single note by ID (GET)
- Update a note (PUT)
- Delete a note (DELETE)

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
